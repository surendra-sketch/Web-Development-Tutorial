JavaScript Type Conversion

JavaScript Data Types

In JavaScript there are 5 different data types that can contain values:

  • string
  • number
  • boolean
  • object
  • function
There are 6 types of objects:
  • Object
  • Date
  • Array
  • String
  • Number
  • Boolean
And 2 data types that cannot contain values:
  • null
  • undefined


The typeof Operator

We can use the typeof operator to find the data type of a javascript variable


The constructor Property

The constructor property returns the constructor function for all JavaScript variables.

Example:

"John".constructor


You can check the constructor property to find out if an object is an Array (contains the word "Array" after using with .constructor property and converted to string using toString)

function checkArray(myArray) {
return (myArray.constructor.toString().indexOf("Array") > -1);
}

Output:


Or even simpler, you can check if the object is an Array function.

function checkArray2(myArray2) {
return myArray2.constructor === Array;
}
